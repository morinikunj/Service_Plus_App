Finding trusted service providers can be time-consuming and confusing. **Multi Services App** solves this by connecting **customers directly to verified service providers** in one platform.  

Customers can browse services, book them instantly, and rate their experience. Service providers can manage their offerings, track bookings, and grow their business. An admin panel ensures smooth management of users, services, and analytics.  

Built with **Flutter** (frontend) and **Node.js** (backend) using REST APIs and JWT authentication.
